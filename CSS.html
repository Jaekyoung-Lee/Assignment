Game

ARPG game(디아블로)

RPG에서 파생된 게임의 장르 중 하나. 이름 그대로 적과 만나서 수동적으로 커맨드만 선택하는 RPG와는 달리 직접 캐릭터를 조작해 턴제나 실시간으로 전투를 치르는 장르 게임

RTS game(스타)

턴제 전략 게임과 구분되는 전략 게임의 하위 장르로 게임상에서 주어진 모든 전략적 요소를 활용하여 적을 없애고 목적을 달성해야 하는 방식의 게임.

하이퍼 FPS game(오버워치)

국내에서만 사용하는 명칭으로 비현실적이거나 속도가 빠른 FPS를 뜻한다.
